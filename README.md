#  led-websocket-server

## setup
```
python3 -m venv env
. ./env/bin/activate
sudo ./env/bin/pip3 install -r requirements.txt
sudo ./env/bin/python led.py
```
* you need sudo in order to use the PWM pin on the raspberry pi
